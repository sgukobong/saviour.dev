## 2025-05-22 - Contact Form Accessibility and UX
**Learning:** Highly custom-styled forms (using Tailwind 'group' patterns) in this app often lack standard accessibility features like `htmlFor` and `id` pairing, and miss clear visual indicators for required fields.
**Action:** Always verify form fields for proper label association and ensure mandatory inputs have visual 'required' indicators using existing design tokens like `brand-rose`.
